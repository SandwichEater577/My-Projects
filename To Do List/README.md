To-do list to aplikacja, która pozwala użytkownikowi dodawać, usuwać i oznaczać zadania jako wykonane. Ten projekt uczy podstaw manipulacji DOM w JavaScript, obsługi zdarzeń i przechowywania danych lokalnie (np. w localStorage). Dzięki niemu można zrozumieć, jak działa interakcja użytkownika z aplikacją oraz jak zarządzać stanem aplikacji. To proste, ale bardzo praktyczne ćwiczenie, które pomaga zbudować fundamenty do bardziej skomplikowanych projektów.
